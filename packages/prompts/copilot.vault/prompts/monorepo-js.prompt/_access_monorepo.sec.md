Start by checking if a template already exists in `.ai/monorepo-template`. If so, this is the desired template.

Otherwise, clone the the repository at:

https://github.com/gregros/monorepo-template

Into the `.ai/monorepo-template` folder.

Study the template's structure, starting with `TEMPLATE_README` and other markdown files which provide documentation regarding the template's structure.

1. vscode configuration
2. typescript configuration
3. package.json configuration
