# Naming conventions
Naming conventions for JavaScript, TypeScript, and Node.js projects:

| Element                              | Convention         |
| ------------------------------------ | ------------------ |
| Class, interface, type, enum         | PascalCase         |
| Function, variable, method, accessor | camelCase          |
| Non-public member                    | Leading underscore |
| Module, class constants              | UPPER_SNAKE_CASE   |
| Environment variable                 | UPPER_SNAKE_CASE   |
| Files, folders                       | kebab-case         |
