# Verbs
General rules are given using capitalized words called Verbs.

## General Verbs
- ALWAYS $given: Always act as $given
- NEVER $given: Never act as $given
- CONSIDER $fact: Think about $given
- PREFER $given: Act as $given UNLESS inappropriate
- AVOID $given: Avoid acting as $given unless appropriate

## Action verbs
### `MUST`
```markdown example
MUST $requirement
MUST refactor the code into classes
```
States a `$requirement` you must complete before the end of your turn.
- NEVER end your turn before completing `$requirement`
- ALWAYS keep track of `$requirement` that you have not finished
