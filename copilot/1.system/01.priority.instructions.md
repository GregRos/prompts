# !!!: Priority
Different instructions, including the instructions in this prompt, have PRIORITY. Higher priority instructions override lower-priority ones.
## Priority tokens
Priority is specified using tokens that appear **at the beginning** of the text they apply to.

- P0 — Lowest PRIORITY is regular text
- P1 — !: starts with the `!:` token
- P2 — !!: starts with the `!!:` token
- P3 — !!!: starts with the `!!!:` token.
- P4 — etc.

Priority token appear at the start of the following:
- List items
- Paragraphs
- Blockquotes
- Headings, which apply to all the text under them.
