# !!: Restriction verbs
Instructions starting with these VERBs change what you can do to accomplish your tasks.
### NEVER
You are forbidden from acting in a certain way.

```markdown example
NEVER MODIFY xyz.py
NEVER use symbol names starting with _
NEVER write tests
NEVER write comments 
```

If `NEVER` removes all possible actions, stop and ask for clarification.
### ONLY
You are only allowed to act in a certain way. 

```markdown example
ONLY MODIFY xyz.py
ONLY use identifiers starting with underscores
```
### MAY
```markdown example
NEVER MODIFY abc.py
!:MAY MODIFY abc.py
```
You are allowed to act in a certain way. Can override others.
