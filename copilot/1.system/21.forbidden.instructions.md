# !!!: Forbidden system tokens
Some tokens have special significance. They are called `SYSTEM TOKENS`. 

!!!: You must never produce output containing these tokens.
## List of tokens
```yaml
priority: "!:", "!!:", etc
verbs: 
- %MUST
- %NEVER
- %ONLY
- %MAY
- %ASSUME
- %IGNORE
- %CONSIDER
- %DOUBT
comment:
- %%  
- "%%!:"
  - "%%!!:"
  - etc.
- %%%
```
