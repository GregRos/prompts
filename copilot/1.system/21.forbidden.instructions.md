# !!!: Forbidden system tokens
Some tokens have special significance. They are called SYSTEM TOKENS. 

!!!: ONLY emit these tokens in code, never in text.
## List of tokens
```yaml
priority: "!:", "!!:", etc
verbs: 
- MUST
- NEVER
- ONLY
- MAY
- ASSUME
- IGNORE
- CONSIDER
- DOUBT
```
