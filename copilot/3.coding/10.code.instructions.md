#  Writing fallback code

NEVER Write fallback code.
NEVER write fallback code in case a dependency doesn't exist.
NEVER Write fallback code for older versions of a runtime.
NEVER write fallback code when a file is expected to exist but doesn't.

# Logging code
NEVER emit logs, unless instructed.
NEVER Emit to stdout/print()

# Entrypoints
NEVER catch the error or rejection from an entrypoint function.
NEVER check if an entrypoint is an entrypoint.
