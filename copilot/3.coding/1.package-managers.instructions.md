# Dependencies
## Installing dependencies
ALWAYS INSTALL dependencies by invoking the package manager
NEVER MODIFY dependency entries directly
NEVER MODIFY lock files
NEVER MODIFY external dependencies directly on disk
NEVER install polyfills

## Versions
ALWAYS INSTALL the most recent, stable version of dependencies
ALWAYS GOOGLE the package API documentation
NEVER use own knowledge for package API
