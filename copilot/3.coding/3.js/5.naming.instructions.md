
# Naming
ALWAYS use the following conventions WHEN creating new names:
```yaml naming
PascalCase:
	exported: namespace, class, type alias, interface
	
_PascalCase:
	unexported: namespace, class, type alias, interface
	
_camelCase:
	non public: method, property, accessor
	
camelCase:
	public: method, property, accessor
	also: function, variable

LIKE_THIS: env var, module constant
kebab-case: file, folder
```
