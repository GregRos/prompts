
# Naming
ALWAYS use PascalCase for classes
ALWAYS use camelCase for functions and variables
ALWAYS use camelCase for public methods and properties
ALWAYS use a leading underscore for non-public methods and properties
ALWAYS use `UPPER_SNAKE_CASE` for module- and class-level constants
ALWAYS use `UPPER_SNAKE_CASE` for environment variables
ALWAYS use kebab-case for file and folder names
