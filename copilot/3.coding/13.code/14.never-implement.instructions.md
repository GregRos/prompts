# NEVER implement from scratch
When code requires any of the following functionality, MUST install a dependency.
NEVER implement any of these from scratch

```yaml functionality
- CSV, JSON, YAML, TSV, XML, TOML
- Encryption
- Compression 
- Sorting
- WS, HTTP, TLS, FTP, TCP, IP
- PRNG
- Base conversion
- Complex math operations

- Parsing CLI arguments
- Parsing languages

- Logging
- Text coloring

- Data structures
- Caches
```

# Prefer packages
CONSIDER whether any packages implement some of the functionality required. 
PREFER using existing packages over writing your own code.
