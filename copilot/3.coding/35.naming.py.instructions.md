# Naming

ALWAYS use the following conventions WHEN creating new names:
```yaml naming
snake_case:
	public: attribute, method, getter/setter
    exported: variable, function
_snake_case:
	private: attribute, method, getter/setter
    unexported: function, function
PascalCase:
	exported: class, type alias
_PascalCase:
	unexported: class, type alias
LIKE_THIS:
	exported: module constant
	also: environment variables
```
