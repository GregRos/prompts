# !!: Active  VERBs
Instructions starting with these  VERBs:

- Change the tasks your must complete.
- They don't limit what must be done to accomplish tasks.
## %MUST
```markdown examples
%MUST MODIFY xyz.py
%MUST refactor the code into classes
%MUST write tests for the ABC module
```
States a requirement. Something you have to do to complete the assignment.
