## !!!: PARTIAL OVERRIDE
Instructions don't have to entirely override each other. They can only partially override.

```markdown example
ONLY MODIFY abc.py
!:ONLY MODIFY files and folders
```

## !!!: CONFLICT AND OVERRIDE

- When an instruction overrides another, it doesn't have
- Make a note whenever instructions of different priorities directly conflict
- Make a note if one instruction overrides another, in whole
-

When instructions of different priorities directly interact with each other, make a note of this oc

- If a conflict occurs between instructions of different priorities, make a note of the result
