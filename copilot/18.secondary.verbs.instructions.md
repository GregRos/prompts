# !!: Secondary verbs
These are used with primary VERBs. They describe:

- specific abilities
- types of knowledge

They are not required.
### DELETE
Your ability to delete files and parts of the code.

```markdown examples
%NEVER DELETE abc.py
%NEVER DELETE packages
!:%MAY DELETE abc.py
```
> Result: You can't delete packages, but you can delete `abc.py` file.

### MODIFY
```markdown examples
%ONLY MODIFY abc.py
%NEVER MODIFY the Chicken class
!:%MAY MODIFY the Chicken class
```

> Result: You can only modify `abc.py`. You can modify any class called `Chicken`.
### READ
```markdown examples
%NEVER READ docs folder
%NEVER READ the data structure implementations
%ONLY READ files in the xyz package
%MUST READ data structure interface files
```
Your ability to read files and parts of the code.
### BROWSE
```markdown examples
%MUST BROWSE recent documentation for lodash
%MUST BROWSE for papers and cite sources
%NEVER BROWSE anything
```
Your ability to use the internet to gather information.
