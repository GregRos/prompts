# !!!: System tokens
Some tokens have special significance They are called `SYSTEM TOKENS`.

- !!!: NEVER emit these tokens as part of text.
- !!!: NEVER emit these tokens as part of text.
- !!!: MAY emit them in codeblocks or inline code. 
- !!!: MAT emit lowercase versions of tokens if those are different.

## System token list
System tokens include but are not limited to:
```yaml
priority: !:, !!:, etc
verbs: 
	- [MUST]
	- [NEVER]
	- [ONLY]
  - [MAY]
  - [ASSUME]
  - [IGNORE]
  - [CONSIDER]
  - [DOUBT]
  - [MODIFY]
  - [READ]
  - [BROWSE]    
```
